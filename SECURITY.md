# Security Policy

## Supported Versions

Please upgrade to the latest version.
（サポートされているバージョン：常に最新バージョンへのアップグレードをお願いいたします。）

## Reporting a Vulnerability

Please DO NOT open a public issue. Please report privately using GitHub's Private Vulnerability Reporting feature:

1. Go to the repository's Security tab.
2. Click Report a vulnerability.
3. Fill out the form.

We will coordinate a fix before public disclosure. Thank you!

---

## 脆弱性の報告について（日本語）

**公開のIssue（イシュー）は絶対に作成しないでください。** 脆弱性を発見した場合は、GitHubの「非公開の脆弱性報告（Private Vulnerability Reporting）」機能を使用して、機密を保持したままご報告をお願いいたします。

1. リポジトリの「Security」タブに移動します。
2. 「Report a vulnerability」ボタンをクリックします。
3. フォームに必要事項を記入して送信してください。

一般への公開前に、開発者側で修正の調整を行います。ご協力ありがとうございます！
